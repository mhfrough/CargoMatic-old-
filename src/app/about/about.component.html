AboutPage
In this System we are making CargoMatic an online service that make it easier for Client to placed package details to deliver form one location to another destination. Client share its request to system about the package details like: Size, Location, Weight, Total in number. Our system get request processes it and delivers the package to desire destination. Pricing will be determined on weight, location distance, how urgent user wants to deliver and where it wants to delivered.
